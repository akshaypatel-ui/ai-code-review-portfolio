# JavaScript/TypeScript Review Expertise

I evaluate JavaScript with a heavy emphasis on security, modern ES6+ syntax, and asynchronous flow control.

## 🔍 Key Pitfalls I Check:
- **Security Vulnerabilities**: Identifying SQL injection (string concat), XSS, and Prototype Pollution.
- **Async Pitfalls**: Spotting `await` in loops, unhandled Promise rejections, and "callback hell."
- **Closure Issues**: Proper scoping and memory leaks in event listeners.
- **Modern Standards**: Enforcing `const`/`let` over `var` and destructuring.
