# Java Review Expertise

My Java reviews prioritize thread safety, robust error handling, and object-oriented best practices (SOLID).

- **Null Safety**: Identifying potential `NullPointerException` (NPE) in un-validated inputs.
- **Resource Management**: Ensuring use of try-with-resources for Streams and DB connections.
- **Precision & Overflow**: Spotting integer overflow in math operations and recursion depth issues.
- **Collections API**: Proper choice of `ArrayList` vs `LinkedList` vs `HashMap`.
